# Search Engine

A full-stack search engine with a C++ backend and React frontend.

## Structure

```
backend/   - C++ search engine core
frontend/  - React UI
docs/      - Architecture and design docs
```

## Getting Started

### Backend
```bash
cd backend
cmake -B build
cmake --build build
./build/search_engine
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
